CookingMasterSimulator
Cooking Master Simulator is a fun and educational VR Cooking game that aims to help novice cooks kickstart their cooking journey in a learner-friendly environment and learn about Asian Cuisine. Set in a virtual kitchen, players can choose from various recipes and follow a set to directions to complete the dish. By exploring different recipes in a gamified kitchen setting, we hope to familiarize players with the navigation of a kitchen, the main methodologies of cooking and important steps of a recipe, all while having fun and reducing waste.
